# phpPDFQRCode
Create a PDF within a QR Code that redirects to the URL of the PDF

# Requirements

* php 5.6

## 1. Clone the App

``` zsh
$ git clone https://github.com/germanngc/phpPDFQRCode.git
```

## 2. Test
``` zsh
$ php -S localhost:8000
```

Open a web browser
